# Character Prediction

The code here is a collection of missing character prediction algorithms.

Example:
    Input: alpha_et
    Output: alphabet

## Evaluation

All algorithms have been evaluated using 10-fold cross validation.

| Algorithm      | Score |
| -------------- | ----- |
| Brute Force    | 0.704 |
| N-grams |  0.55  |
| Regex   |  0.69  |
